  接口说明

一键快传分类列表，或者去掉该接口，分类内容固定在代码里。

 

参数名称

参数类型

参数说明

输入值类型

String

用户token

CompanyId

String

用户companyId

 

返回值类型

      说明

String

      说明

参数类型

参数说明

CompanyId
